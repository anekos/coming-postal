# coming-postal

Fetch and display tracking information.

## Usage

```
$ java -jar coming-postal-0.1.0-SNAPSHOT-standalone.jar LO123456789CN
2019/12/01 20:20 - 引受 -
2019/12/02 10:10 - 国際交換局に到着 -
2019/12/04 01:23 - 国際交換局から発送 -
2019/12/14 12:23 - 国際交換局に到着 -
```

## Supported services

- Japan Post
- Kuroneko Yamato
- Sagawa
